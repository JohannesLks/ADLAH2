# Rollen-Verzeichnis

Hier kommen die einzelnen Rollen für dein Ansible-Projekt hinein. Jede Rolle erhält einen eigenen Unterordner.

Beispielstruktur für eine Rolle:

roles/
  beispielrolle/
    tasks/
      main.yml
    handlers/
      main.yml
    files/
    templates/
    vars/
      main.yml
    defaults/
      main.yml
    meta/
      main.yml
